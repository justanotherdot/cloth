# Cloth

Feature flag management on Cloudflare Workers.

## Development

```sh
npm install
npm run dev
```

## Testing

```sh
npm test
```

## Deployment

```sh
npm run deploy
```

## Scripts

- `bin/check` - Type checking
- `bin/test` - Run tests
- `bin/format` - Fix formatting
- `bin/format:check` - Check formatting
- `bin/lint` - Check linting
- `bin/lint:fix` - Fix linting
